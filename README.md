## 🚀 Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repo:

   ```bash
   git clone https://github.com/amitabhkr/portfolio.git
   ```

2. Navigate to the project directory:

   ```bash
   cd portfolio
   ```

3. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

### Running the Development Server

```bash
npm start
# or
yarn start
```

The development server will start at `http://localhost:3000`.
